## Abstract
本研究は、東日本大震災および福島第一原子力発電所事故によって甚大な被害を受けた福島県双葉郡楢葉町を対象に、地理情報可視化プラットフォームRe:Earthに含まれる可視化機能であるRe:Earth Visualizerを用いる。デジタル災害アーカイブの構築を目的とする。  
震災から十数年が経過し、当時の経験を直接語ることのできる世代が限られつつある現在、記憶の風化を防ぎ、震災の教訓を次世代へ継承するための仕組みづくりが強く求められている。本研究では、被災者や地域住民へのインタビューおよび現地観察によって得られた写真・映像・証言・地図情報などの資料を収集・整理し、空間情報と統合したインタラクティブな災害アーカイブのプロトタイプ制作を試みる。  
楢葉町に特化することで、被災直後から復興に至るまでの過程を時系列かつ空間的に可視化し、防災教育や地域学習への活用可能性を検討する点に本研究の特徴がある。また、地域固有の経験を空間情報と結び付けて記録することで、出来事と場所の関係性を直感的に理解できる点にも意義が見いだされる。  
さらに、本研究の作成者自身が福島県出身であり、震災当時の状況を実体験として語ることのできる世代であるという立場から、当事者としての視点と責任を意識し、記録と発信を通じて復興に主体的に関わる姿勢をもって本研究に取り組んでいる点も、本研究の重要な背景であり、研究動機の中核をなしている。  
## Introduction
東日本大震災および福島第一原子力発電所事故は、福島県を中心とする地域社会に深刻かつ長期的な影響を及ぼした。震災から十数年が経過した現在、当時の出来事を自らの経験として語ることのできる世代は限られつつあり、記憶の風化が進んでいる。こうした状況に対し、福島県出身であり震災を語れる最後の世代の一人である筆者は、記録と発信を通じて復興に貢献したいという思いを抱いてきた。  
特に福島県双葉郡楢葉町は、比較的早期に避難指示が解除され、被災から復興に至る過程を継続的に追うことが可能な地域である。一方で、既存の災害アーカイブは広域的なものが多く、地域固有の経験や変化を体系的に可視化した事例は十分とは言えない。  
そこで本研究では、地理情報可視化プラットフォームRe:Earthを用い、楢葉町における震災記録を空間情報と結び付けて整理・発信するデジタル災害アーカイブの構築に取り組む。同様の災害が再び起こった際に、東日本大震災の記録を教訓として活かすためのツールの一つとなることを目指す。このような立場性を踏まえ、学術的検討と社会的実践を両立させる研究として本テーマを設定した。  
## Methods
本研究では、資料収集に先立ち、Re:Earth Visualizerを用いた災害アーカイブのプロトタイプ作成を中心に方法設計を行った。災害デジタルアーカイブは、単なる資料保存にとどまらず、利用者が主体的に災害を理解し、記憶を継承するための場として設計されることが重要であると指摘されている（佐藤ほか，2019；江草ほか，2020）。本研究では、こうした先行研究の知見を踏まえ、Re:Earth Visualizerを用いた空間的可視化に重点を置いたプロトタイプ構築を方法として採用した。  
初期段階では、Re:Earth Visualizerの機能理解が不十分であり、特にCSVデータのインポート作業において実装上の課題に直面した。そこで、Re:Earth Visualizerの開発・保守を行う日本のソフトウェア企業Eukaryaが主催するハンズオンセミナーに参加し、基本操作や利用可能な表現方法を学ぶことで、ツールに対する理解度の向上を図った。  
その後、楢葉町に実際に足を運び、現地での移動を通じてStravaを用いた地形データの記録を行った。取得した軌跡データはGeoJSON形式へ変換し、Re:Earth Visualizerへ読み込むことで、地形や移動経路を反映した可視化を試みた。これにより、現地観察に基づく空間データを用いた表現の検証を行い、今後のアーカイブ構築に向けた技術的基盤を整えた。加えて、データ構造や階層設計の検討も並行して行った。  
## Result
本研究は現在進行段階にあり、ここでは現時点までに得られた結果を整理する。  
Re:Earth Visualizerの操作理解については、Eukarya主催のハンズオンセミナーに参加したことで、CSVやGeoJSONデータの構造や読み込み方法を把握し、可視化作業を自力で進められる段階に到達した。さらに、楢葉町においてStravaを用いて取得した地形データをGeoJSON形式へ変換し、Re:Earth Visualizer上に反映した結果、現地での移動経路や地形的特徴を空間的に把握できることが確認された。  
一方で、データの階層構造やレイヤー設計によって可読性が大きく左右されることも明らかとなり、設計段階の重要性が示唆された。  
今後の研究段階においては、被災者や地域住民、関係者へのインタビュー調査を実施し、震災当時の状況や復興過程に関する証言を収集することを予定している。あわせて、「ならはCANvas」を活用した現地観察を行い、写真・映像・証言・地図情報などの一次資料を取得することで、Re:Earth Visualizer 上に構築した災害アーカイブの内容をより実証的かつ多層的なものへと発展させていくことを目指す。  
## Discussion
本研究で得られた現段階までの成果から、地域特化型のデジタル災害アーカイブは、広域的な災害記録では把握しにくい地域固有の変化や復興の過程を可視化する手法として有効であることが示唆された。先行研究では、災害デジタルアーカイブが記憶の固定化ではなく、利用者との相互作用を通じて意味づけられる場として機能することの重要性が示されている（佐藤ほか，2018；江草ほか，2020）。本研究で明らかになったRe:Earth Visualizerにおける階層構造や視覚表現の影響は、こうした議論を空間情報の観点から具体化する試みとして位置付けることができる。特に楢葉町のように避難指示解除後も復興が継続している地域では、被災直後の状況だけでなく、その後の再生の歩みを含めて記録することに大きな意義があると考えられる。  
一方で、現時点では資料内容が地形データや既存情報に限定されており、視点の偏りが生じる可能性がある点は課題である。また、Re:Earth Visualizer の操作性や表現方法には一定の制約があり、情報の見せ方や理解のしやすさがアーカイブの質に影響することも明らかになった。  
今後は、被災者や地域住民へのインタビュー、ならはCANvasでの一次資料収集を通じて内容を充実させるとともに、防災教育や地域学習の場においてどのような表現や構成が有効かを利用者視点から検討していく必要がある。  
## Conclusion  
本研究の一連の取り組みを通じて、地理情報と証言や映像資料を組み合わせたデジタル災害アーカイブが、震災の記憶を空間的かつ立体的に理解するうえで有効な手法であることが確認された。楢葉町に焦点を当てたことで、被災直後の状況だけでなく、復興に向けた時間的な変化や地域の再生過程を具体的に示す可能性が明らかとなった。この点は、広域的な災害記録では捉えにくい地域固有の経験や文脈を残す意義を示している。  
一方で、現段階の成果は主に地形データや既存資料に基づくものであり、今後は被災者や地域住民、関係者へのインタビュー、ならはCANvasでの現地観察を通じた一次資料の追加が不可欠である。これらの作業を進めることで、記録内容の厚みと信頼性の向上が期待される。また、Re:Earth Visualizerの操作技術や表現方法をさらに習熟させ、情報構造や視覚的整理を改善することで、より完成度の高いプロトタイプの構築が求められる。  
将来的には、地域住民や教育機関からのフィードバックを反映し、防災教育や地域学習に活用可能な災害アーカイブへと発展させることが本研究の到達点であるといえる。
