## Abstract
本研究は、東日本大震災および福島第一原子力発電所事故により甚大な被害を受けた福島県双葉郡楢葉町に特化したデジタル災害アーカイブを構築することを目的として進められている。地理的情報可視化プラットフォームである Re:Earthを活用し、震災に関する写真や映像、証言、地図データを統合したインタラクティブなアーカイブのプロトタイプ制作を目指す。震災から14年が経過し、記憶の風化が進むなか、震災を語ることのできる最後の世代として記録と発信を持続していく意義を基盤として研究を進めている。
## Introduction
2011年3月11日に発生した東日本大震災と、それに続く福島第一原子力発電所事故は、福島県双葉郡に甚大な被害をもたらした。地震・津波・放射能汚染という複合的な災害は、人々の生活基盤を根底から揺るがし、多くの住民が長期的な避難生活を余儀なくされた。双葉郡の中でも楢葉町は比較的早期に避難指示が解除され、住民の帰還が徐々に進んだ地域であり、震災直後から復興に至る過程を記録することは学術的にも地域的にも重要な意味を持つ。  
震災から14年が経過し、当時の記憶を直接語ることのできる世代が限られつつある今、地域の出来事を記述し続けることは、記憶の継承と防災教育の両面で欠かせない。本研究は、楢葉町に特化したデジタル災害アーカイブを構築することを目的とし、地理情報システム（GIS）とデジタルアーカイブ技術を統合できる Re:Earth を主要なプラットフォームとして採用する。  
広島アーカイブや長崎アーカイブ、東日本大震災アーカイブなどの先行事例は、証言・写真・地図など多様な資料を統合するデジタルアーカイブが、過去の災害を可視化し、記憶の継承に大きな役割を果たすことを示してきた。しかし楢葉町については、同様の形式で体系化された地域固有のアーカイブが十分に整備されていない。  
そこで本研究では、被災者の証言、被害状況や復興過程を示す写真・動画、現地観察データなど多様な資料を収集し、Re:Earth 上で空間情報と統合して可視化することで、地域の震災史を多層的に記録する試みを行う。また、このアーカイブを地域教育や防災学習に活かすことを視野に入れ、その利用可能性を検討する。  
## Methods
・被災者・地域住民・関係者へのインタビューを行い、震災当時の状況や復興に関する記録を収集する。  
・「ならはCANvas」での現地観察を通じて、写真・動画・メモ・地図情報など、アーカイブに必要な資料を追加収集する。  
・収集した資料に対して、撮影日・撮影場所・内容説明などのメタデータを付与し、整理・デジタル化を進める。  
・Re:Earthにデータを統合し、空間情報と資料を多層的に重ね合わせたインタラクティブマップとしてプロトタイプを構築する。  
・防災教育・地域学習での利用を想定し、アーカイブとしての活用可能性、操作性、改善点などを検討する。
